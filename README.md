# Sean Troxel — Professional Website

Personal professional/resume-style website, hosted on GitHub Pages at [troxelscode.github.io](https://troxelscode.github.io/).

## Tech stack

Plain HTML, CSS, and JavaScript — no build tools, frameworks, or package manager.

## Local development

Serve the directory with a local static server and open the printed URL in a browser:

```
python -m http.server
```

Alternatively, open [index.html](index.html) directly in a browser.

## Deployment

GitHub Pages builds automatically from the `main` branch root on every push — no separate deploy step.

## Project structure

```
index.html      Entry point
css/style.css   Stylesheet
js/main.js      Script entry point
```
